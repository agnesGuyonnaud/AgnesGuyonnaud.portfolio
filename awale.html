Awale Game Server

Client/Server application to play an african strategy game, developped in C using sockets.

We Implemented the Awale game : its rules, counting of points and displaying of board.

But the main purpose of the project was to build a client/server application, enabling each client to register with a username, challenge a fellow user and of course play the match. But they can also observe a match between two other users

Each user can also write a bio, add friends and play in public or private mode, preventing users that are not the player's friend to observe the match.

We decided to add an anytime chat fonctionnality, so that a client can send a message to any other client, or to all a game table (playes and observers).
